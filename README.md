# Обработка изображений (Вариант 10)

Приложение с графическим интерфейсом для бинаризации изображений, выделения контуров объектов и отображения их габаритов.

## Функциональность

- Загрузка изображений (JPG, PNG, BMP).
- Отображение оригинала на первой вкладке.
- На второй вкладке:
  - Ползунок регулировки порога яркости (0–255).
  - Автоматическое применение бинарного порога (cv2.threshold).
  - Поиск контуров (cv2.findContours) и отсев мелких объектов (площадь < 100 пикселей).
  - Отображение ограничивающих прямоугольников (cv2.boundingRect).
  - Подсчёт количества найденных объектов.
- Сохранение результата (изображение с прямоугольниками) в файл.
